安装

使用vagrant时，会启动虚拟机，默认使用的虚拟机是virtualbox，所以使用vagrant前，首先安装virtualbox和vagrant

同步的目录

默认设置下，上面的工作环境根目录E:\projects\vagrant-project1与虚拟机中的目录/vagrant是一致的，始终同步，可以在虚拟机中使用命令ls /vagrant查看目录文件，内容与主机开发环境根目录完成一样。主机开发环境根目录下对文件的所有读写操作，都会同步到虚拟机的/vagrant中

可以修改vagrantfile文件，添加更多同步目录

基本用法

  • 启动虚拟机
    vagrant up

  • 登录虚拟机
    vagrant ssh
    Windows命令行中没有安装ssh client的话，请使用putty

  • 销毁虚拟机
    vagrant destory

  • 关闭虚拟机
    vagrant halt

  • 挂起虚拟机
    vagrant suspend

  • 恢复虚拟机
    vagrant resume
